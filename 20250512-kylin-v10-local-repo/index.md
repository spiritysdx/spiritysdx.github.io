# 麒麟 V10 安装时配置本地源


麒麟 V10 安装时如果没有正确指定本地镜像源，安装界面可能报错：

```text
设置基础软件仓库时出错
```

这个问题需要在进入图形安装界面之前处理，也就是在启动引导界面修改内核参数。

## 操作步骤

在安装引导界面，将光标移动到：

```text
Install Kylin Advanced Server V10
```

按 `Tab` 或 `e` 进入启动参数编辑页面。

在启动参数末尾追加：

```text
inst.repo=hd:/dev/sdb1
```

其中 `/dev/sdb1` 需要与原有参数中的 `inst.stage2=hd:/dev/sdb` 保持同一块安装介质或分区来源。实际设备名可能是 `/dev/sda1`、`/dev/sdb1` 或其他值，需要按现场环境调整。

修改完成后按回车继续进入安装 UI。此时安装器会把指定介质作为本地软件源，后续即可继续安装。

## 注意事项

- 这个参数必须在进入安装 UI 前追加，进入 UI 后再改通常无法补救。
- `inst.repo` 指向的是安装源，`inst.stage2` 指向的是安装阶段镜像，两者设备路径应保持一致。
- 如果仍然报错，优先检查 U 盘或 ISO 挂载设备名是否发生变化。
